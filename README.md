
第一步，将问渠学堂中的课程学习页面本地保存html
	1.同样打开目标网页 → 按 F12 打开开发者工具。（笔记本或者小键盘按FN+F12）
	2.切换到「检查器」标签（小鼠标箭头那个）。
	3.在左边代码区，最上面会高亮显示 <html>。
	4.右键点击这个 <html> 节点。
	5.选择：Copy → Outer HTML（中文版显示：复制 → 外部 HTML）
	6.同样打开记事本 → 粘贴 → 文件 → 另存为 → 文件名写 .html → 编码 UTF-8 → 保存。
（验证你保存的是不是正确的完整文件双击你刚保存的 .html 文件，它会用浏览器打开，和原来网页一模一样（图片都在），说明保存成功了！）

第二步，打开wqxt_ppt.py，修改第七行代码‘html_file = r"D:\xxxxxx\xxxxx\xxxxxx\saved_page.html"   # ← 改这里’。
	1.改为第一步.html文件的保存地址。
	2.然后运行。
	3.会在程序所在当前文件夹生成名为‘PPT全套’的文件夹。
第三步，打开并运行tongyicaijian.py,找到‘PPT全套’并打开，用鼠标开始对第一张ppt进行统一裁剪。会在‘PPT全套’里生成一个名为‘裁剪后’的文件夹。
第四步，打开并运行pdf.py,选择‘裁剪后’并打开，在命令行输入3或4（想在一张a4张装3张图片还是4张图片）然后回车。再按一下回车结束程序。
